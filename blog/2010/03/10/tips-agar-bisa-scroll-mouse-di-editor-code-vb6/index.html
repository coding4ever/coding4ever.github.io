Tips Agar Bisa Scroll Mouse Di Editor Code Vb6

Mar 10th, 2010 7:01 am | Comments

Ini mungkin tips lama tapi saya sendiri juga baru tau he he he :), maklum soalnya udah ngerasa cukup dengan editor codenya vb6 walaupun kadang-kadang cukup mengganjal dihati klo lagi coding di java menggunakan eclipse/netbeans.

Benar-benar perbedaan editor code yang terlalu mencolox :D salah satunya kekurangannya adalah di editor code vb 6 enggak bisa scroll mouse, tapi dengan tips berikut kekurangan ini bisa kita tutupi.

Oke berikut langkah-langkahnya :

1. Tutup dulu program VBnya

  1. Download Add-In MouseWheel Fix disini kemudian ekstrak di lokasi yang Anda inginkan, misal c:\MouseWheel

  1. Setelah itu langsung menuju ke TKP kemudian copykan file VB6IDEMouseWheelAddin.dll ke windows\system32 (jangan lupa diregister).

  2. Jalankan lagi program VBnya, kemudian klik menu Add-Ins -> Add-In Manager …

  1. Aktifkan Add-In MouseWheel Fix

  1. Selesai

Selamat mencoba :)
